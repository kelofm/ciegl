Detailed Description

Immutable uniform descriptor (for any shader).

Represents data global to all entities of the shader type (vertices for vertex shaders, fragments for fragment shaders, etc.). For example: transformation matrix, background color, etc. Requires:

  • name : name of the uniform in the shader code
  • size : number of components that make up the uniform
  • type : data type
