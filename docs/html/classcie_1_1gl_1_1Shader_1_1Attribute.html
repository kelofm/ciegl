Detailed Description

Immutable attribute descriptor for vertex shaders.

Represents an attribute passed to each vertex. For example: position, texture coordinates, etc. Requires:

  • name : name of the variable in the shader (eg: position)
  • size : number of components that make up the attribute (eg: x,y,z -> 3)
  • stride : number of bytes in between the components of the attribute (that are not part of the attribute itself)
  • offset : number of bytes before the first component of the attribute
